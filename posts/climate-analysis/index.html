📌 Introduction

The NOAA Climate Data provides an extensive record of global temperature trends, collected from thousands of weather stations across various geographic regions. This dataset is crucial for analyzing long-term climate patterns, regional warming trends, and the impacts of climate change.

In this blog, we will explore how to:

✔ Construct and query an SQL database for efficient climate data retrieval
✔ Apply filtering techniques to extract relevant temperature data
✔ Compute meaningful temperature trends over time
✔ Visualize climate patterns using advanced interactive charts

By the end, you will be able to analyze global and regional temperature changes efficiently using SQL and Python.

🌍 1. Creating the Climate Database

Our first objective is to create an SQL database containing three data tables:
📌 temperatures (historical temperature records)
📌 stations (weather station metadata)
📌 countries (geographic mappings)

We will start by downloading the NOAA climate data and structuring it into an SQLite database.

📥 1.1 Downloading NOAA Temperature Data

The NOAA dataset is split into decadal CSV files (e.g., 1901-1910.csv, 1911-1920.csv, etc.), which we need to retrieve from the GitHub repository.

💾 1.2 Creating Our SQL Database

After downloading the NOAA climate dataset, we must now convert the raw data into a structured SQL database.

This process involves:

  1. Cleaning the data and converting it into a query-friendly format.
  2. Storing it in an SQLite database.
  3. Adding necessary metadata, such as weather station and country mappings.

📂 1.2.1 Structuring Temperature Data

The NOAA dataset stores temperature records in wide format, meaning each row contains temperature readings for all 12 months of a given year. However, for better SQL queries, we need a long format, where each row represents a single temperature reading for a specific month and year.

We accomplish this transformation with the following function:

import sqlite3
import pandas as pd
import numpy as np

def prepare_df(df):
    """
    Converts a wide-format temperature dataset into a long-format dataset.
    """
    df = df.melt(
        id_vars=["ID", "Year"],
        value_vars=[f"VALUE{i}" for i in range(1, 13)],
        var_name="Month",
        value_name="Temp"
    )
    df["Month"] = df["Month"].str.extract("(\d+)").astype(int)
    df["Temp"] = df["Temp"] / 100  # Convert temperatures to Celsius
    df = df.dropna(subset=["Temp"])  # Remove NaN values
    return df

🛠 1.2.2 Storing Data in SQLite

Now that we have transformed our temperature data into a long-format structure, we will store it in an SQLite database.

This database will contain three tables: 1. temperatures → Contains historical temperature records. 2. stations → Metadata about weather stations. 3. countries → Country mappings for station identification.

The temperature dataset is split into decadal CSV files (e.g., 1901-1910.csv, 1911-1920.csv, etc.).

The script below:

  • Reads each decade’s data.
  • Formats the dataset using our prepare_df() function.
  • Stores it into the database under the temperatures table.
# Define decade-based intervals
intervals = [f"{i}-{i+9}" for i in range(1901, 2020, 10)]

# Insert temperature data into SQLite
for i, interval in enumerate(intervals):
    filepath = f"datafiles/{interval}.csv"
    df = pd.read_csv(filepath)  # Load CSV file
    df = prepare_df(df)  # Convert to long format

    # Insert into SQLite database
    df.to_sql("temperatures", conn, if_exists="replace" if i == 0 else "append", index=False)

🔍 2. Querying the Climate Database

Now that our SQL database is set up, we define a function to retrieve climate records dynamically. Instead of writing SQL queries manually each time, we create a reusable function.

🛠 2.1 Writing a Query Function

We define query_climate_database() to retrieve temperature readings for a specific country, year range, and month.

import sqlite3
import pandas as pd

def query_climate_database(db_file, country, year_begin, year_end, month):
    """
    Queries the climate database for temperature readings in a specific country, year range, and month.
    """
    conn = sqlite3.connect(db_file)
    
    query = f"""
        SELECT 
            S.Name AS NAME,
            S.LATITUDE AS LATITUDE,
            S.LONGITUDE AS LONGITUDE,
            C.Name AS Country,
            T.Year AS Year,
            T.Month AS Month,
            T.Temp AS Temp
        FROM 
            temperatures T
        JOIN 
            stations S ON T.ID = S.ID
        JOIN 
            countries C ON SUBSTR(S.ID, 1, 2) = C."FIPS 10-4" 
        WHERE 
            C.Name = '{country}' AND
            T.Year BETWEEN '{year_begin}' AND '{year_end}' AND
            T.Month = '{month}'
        ORDER BY
            S.Name ASC
    """
    
    df = pd.read_sql_query(query, conn)
    conn.close()
    return df

✅ Function Features:

  • Retrieves temperature data for a given country and time period.
  • Joins stations and countries to match locations.
  • Filters data based on user input.

📍 3. Visualizing Climate Data

In this section, we use Plotly to create interactive visualizations of temperature trends. ## 📌 3.1 Yearly Average Temperature Trends{.unnumbered}

This visualization shows how average temperatures in a country change over time.

import plotly.express as px
import sqlite3
import pandas as pd

def faceted_yearly_avg_temp_plot(db_file, countries):
    """
    Creates a faceted line plot comparing yearly average temperature trends across multiple countries.

    Parameters:
    - db_file (str): Database file name.
    - countries (list): List of country names to compare.

    Returns:
    - A Plotly faceted figure.
    """
    conn = sqlite3.connect(db_file)
    
    query = f"""
        SELECT Year, C.Name AS Country, AVG(Temp) AS Avg_Temp
        FROM temperatures T
        JOIN stations S ON T.ID = S.ID
        JOIN countries C ON SUBSTR(S.ID, 1, 2) = C."FIPS 10-4"
        WHERE C.Name IN ({", ".join(["'"+c+"'" for c in countries])})
        GROUP BY Year, C.Name
        ORDER BY Year
    """
    
    df = pd.read_sql_query(query, conn)
    conn.close()

    # Compute rolling averages
    df["Rolling_Avg_Temp"] = df.groupby("Country")["Avg_Temp"].transform(lambda x: x.rolling(window=5, min_periods=1).mean())

    # Create faceted plot
    fig = px.line(
        df,
        x="Year",
        y=["Avg_Temp", "Rolling_Avg_Temp"],
        color="Country",
        facet_col="Country",
        title="Yearly Average Temperature Trends Across Multiple Countries",
        labels={"Avg_Temp": "Avg Temp (°C)", "Rolling_Avg_Temp": "5-Year Rolling Avg"},
        markers=True
    )

    return fig

# 📊 Generate faceted plot for India, United States, and Germany

This plot illustrates the temperature variation across different months in a given year.

🔍 Understanding Monthly Temperature Variations

Temperature patterns fluctuate throughout the year, influenced by seasonal changes. To visualize this, we use violin plots that display the distribution of temperature values for each month.

Key Insights: - We observe monthly temperature variations to identify seasonal trends. - The plot uses violin distributions to show the spread of temperatures. - A boxplot overlay highlights median values and interquartile ranges.

📌 Querying Temperature Data for a Given Year

The function below extracts monthly temperature data for a specified country and year.

def monthly_temp_distribution(db_file, country, year):
    """
    Creates an interactive violin plot with a box overlay for monthly temperature distributions.

    Parameters:
    - db_file (str): Database file name.
    - country (str): Target country.
    - year (int): Specific year for analysis.

    Returns:
    - A Plotly figure object.
    """
    conn = sqlite3.connect(db_file)

    query = f"""
        SELECT Month, Temp
        FROM temperatures T
        JOIN stations S ON T.ID = S.ID
        JOIN countries C ON SUBSTR(S.ID, 1, 2) = C."FIPS 10-4"
        WHERE C.Name = '{country}' AND T.Year = {year}
    """

    df = pd.read_sql_query(query, conn)
    conn.close()

    # Create violin plot to show temperature distribution
    fig = px.violin(
        df,
        x="Month",
        y="Temp",
        box=True,
        points="all",
        title=f"Monthly Temperature Distribution in {country} ({year})",
        labels={"Temp": "Temperature (°C)"},
    )

    return fig

# Generate and display the figure

fig2.show()

🌍 3.3 Temperature Trend Map

We estimate yearly temperature trends at each station using linear regression. This helps us analyze regional climate variations over time by identifying long-term warming or cooling trends.

🔢 Computing Temperature Trends

To estimate temperature trends, we fit a linear regression model for each station, where the year is the predictor and the temperature is the response variable.

import warnings
warnings.filterwarnings("ignore")
import plotly.express as px
from sklearn.linear_model import LinearRegression
import numpy as np

def temperature_coefficient_plot(db_file, country, year_begin, year_end, month, min_obs=10, **kwargs):
    """
    Creates an interactive geographic scatterplot for yearly temperature trends.

    Parameters:
    - db_file (str): Database file name.
    - country (str): Target country.
    - year_begin (int): Start year.
    - year_end (int): End year.
    - month (int): Month for analysis.
    - min_obs (int): Minimum number of observations required per station.
    - **kwargs: Additional arguments for px.scatter_mapbox().

    Returns:
    - A Plotly interactive map.
    """
    df = query_climate_database(db_file, country, year_begin, year_end, month)

    # Ensure each station has enough data points
    df_count = df.groupby("NAME")["Year"].transform("count")
    df = df[df_count >= min_obs]

    # Compute temperature trend using linear regression
    def compute_slope(group):
        if len(group) < min_obs:
            return np.nan
        X = group[["Year"]].values.reshape(-1, 1)
        y = group["Temp"].values
        model = LinearRegression().fit(X, y)
        return model.coef_[0]  # Extract the slope (temperature trend)

    # Apply function to each station group
    station_trends = df.groupby("NAME").apply(lambda g: compute_slope(g)).reset_index()
    station_trends.columns = ["NAME", "Temp_Trend"]

    # Merge with station locations
    df = df.drop_duplicates(subset=["NAME"])
    df = df.merge(station_trends, on="NAME")

    # Create interactive scatter plot using Plotly
    fig = px.scatter_mapbox(
        df,
        lat="LATITUDE",
        lon="LONGITUDE",
        color="Temp_Trend",
        hover_name="NAME",
        hover_data={"NAME": True, "Temp_Trend": ":.3f"},
        title=f"Yearly Temperature Trend in {country} ({year_begin}-{year_end})",
        labels={"Temp_Trend": "Temperature Change (°C/yr)"},
        mapbox_style="carto-positron",
        zoom=2,
        **kwargs
    )

    # Set color scale for better visualization
    fig.update_layout(
        coloraxis_colorbar_title="Temperature Trend (°C/yr)",
        margin={"r":0,"t":40,"l":0,"b":0}
    )

    return fig

# 🌍 Generate scatterplot for India (1980-2020)

fig1.show()
