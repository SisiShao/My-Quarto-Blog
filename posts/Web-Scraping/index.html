Introduction

Wouldn‚Äôt it be nice to find more shows similar to your favorite ones? Many modern platforms, such as Netflix and Spotify, use recommender systems to suggest content.

In this post, we will use web scraping to create a simple recommendation system based on shared actors between movies and TV shows.

We will scrape data from The Movie Database (TMDB) using the scrapy library.

2Ô∏è‚É£ Implementing the Scraper

We will implement a three-step scraper to:

  1. Extract actor links from a movie‚Äôs cast page.
  2. Visit each actor‚Äôs profile and extract their acting roles.
  3. Filter out non-acting roles and store only relevant movies & TV shows.

üìú tmdb_spider.py (Scrapy Code)

import scrapy

class TmdbSpider(scrapy.Spider):
    name = "tmdb_spider"

    def __init__(self, subdir="", *args, **kwargs):
        """Initialize the spider with a specific movie subdirectory from TMDB."""
        super().__init__(*args, **kwargs)
        self.start_urls = [f"https://www.themoviedb.org/movie/{subdir}/"]

    def parse(self, response):
        """Step 1: Navigate to the Full Cast & Crew page of the movie."""
        cast_url = response.url.rstrip("/") + "/cast"
        yield scrapy.Request(url=cast_url, callback=self.parse_full_credits)

    def parse_full_credits(self, response):
        """Step 2: Extract all actor profile links and visit each actor's page."""
        actors = set(response.css("ol.people.credits:not(.crew) li a::attr(href)").getall())  # Exclude crew members
        for actor in actors:
            #need callback=self.parse_actor_page to tell Scrapy to wait for the response before calling the function
            yield response.follow(actor, callback=self.parse_actor_page)

    def parse_actor_page(self, response):
        """Step 3: Extract the actor's name and their acting roles (only from the first Acting table)."""
        actor_name = response.css("h2.title a::text, h1.title::text").get()

        movies = set(response.xpath(
            "//h3[contains(translate(text(), 'ABCDEFGHIJKLMNOPQRSTUVWXYZ', 'abcdefghijklmnopqrstuvwxyz'), 'acting')]"
            "/following-sibling::table[1]//td[contains(@class, 'role')]/a/bdi/text()"
        ).getall())

        for movie in movies:
            yield {"actor": actor_name.strip() if actor_name else "Unknown", "movie_or_TV_name": movie.strip()}

üìù Parsing Methods Explained

1. parse()

The parse() method is responsible for extracting actor links from a movie‚Äôs cast page. It assumes that the response is a movie page and uses CSS selectors to locate all cast members. Once the actor links are found, it follows each link and passes the response to parse_full_credits(), ensuring that the scraper retrieves detailed information about each actor‚Äôs filmography.

2. parse_full_credits()

The parse_full_credits() method extracts the full list of movies and TV shows that an actor has appeared in. It assumes that the response comes from an actor‚Äôs profile page and navigates through the filmography section using CSS selectors. Each extracted role is then passed to parse_actor_page(), allowing the scraper to process only relevant acting roles and filter out other credits such as directing or producing.

3. parse_actor_page()

The parse_actor_page() method processes the details of a movie or TV show that an actor has been involved in. It assumes that the response corresponds to a movie or TV show page and extracts the title along with the actor‚Äôs name. Additionally, it verifies whether the role is an acting role by checking the role description on the page. If the role qualifies, it yields a dictionary containing the actor‚Äôs name and the corresponding movie or TV show title, ensuring that only acting credits are stored in the final dataset.

3Ô∏è‚É£ Running the Scraper

To run the scraper for a specific movie like ‚ÄúHarry Potter and the Philosopher‚Äôs Stone‚Äù, use:

scrapy crawl tmdb_spider -o results.csv -a subdir=671-harry-potter-and-the-philosophe

This will generate a CSV file with two columns:

  • actor: Name of the actor
  • movie_or_TV_name: Movies or TV shows they have appeared in

4Ô∏è‚É£ Making Recommendations

Now that we have the data, we can process it to find movies and TV shows that share actors with our favorite movie.

üìä Analyzing Shared Actors

We use Pandas to count and sort the movies with the most shared actors:

Code 
import pandas as pd

# Load scraped data
df = pd.read_csv("results.csv")

# Count occurrences of each movie/TV show
recommendations = df['movie_or_TV_name'].value_counts().reset_index()
recommendations.columns = ['Movie/TV Show', 'Number of Shared Actors']

üìà Visualization

To better understand the recommendations, we can create a bar chart:

Code 
import matplotlib.pyplot as plt

# Plot top recommendations
plt.figure(figsize=(10, 5))
plt.barh(recommendations['Movie/TV Show'][:10], recommendations['Number of Shared Actors'][:10])
plt.xlabel("Number of Shared Actors")
plt.ylabel("Movie/TV Show")
plt.title("Top 10 Movies/TV Shows with Shared Actors")
plt.gca().invert_yaxis()
plt.show()
